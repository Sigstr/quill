# Tag
## Summary
Tags are used to label the status of an entity. If anything other than green or blue is passed as the theme, the tag will be styled grey.

## Props
<Props of={Tag} />

## Examples

### Grey theme
<Playground>
  <Tag>Inactive</Tag>
</Playground>

### Green theme
<Playground>
  <Tag theme="green">Active</Tag>
</Playground>

### Blue theme
<Playground>
  <Tag theme="blue">Upcoming</Tag>
</Playground>
