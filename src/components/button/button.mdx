# Button

## Summary

Buttons used throughout the UI. There are three types of Buttons, all controllable by css `className`.

- Default : `.button`
- Contained : `.button.button-contained`
- Outlined : `.button.button-outlined`

## Examples

### Default Button

    className="button"
