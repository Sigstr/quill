# Button
## Summary
Use buttons as actions. Sometimes, buttons expose more actions in cases like the action menu, which must be exposed by clicking somewhere.

 There are three types of Buttons, all controllable by css `className`.

- Default : `.button`
- Contained : `.button.button-contained`
- Outlined : `.button.button-outlined`
