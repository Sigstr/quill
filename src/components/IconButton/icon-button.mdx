# Icon Button

## Summary

Use Icon Buttons as actions. Sometimes, icon buttons expose more actions in cases like the action menu, which must be exposed by clicking somewhere.

## Example

<Playground>
  <IconButton
    onClick={() => alert("You clicked the icon button")}
  />
