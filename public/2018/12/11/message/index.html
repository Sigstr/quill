Summary

Use messages when communicating conditions or indicating an event. Content can include authentication, information, confirmation, warnings, and error messages. It is important to specify which of these the content of the message falls under in order to choose the colors

Modifiers

  • Width: (Defaults to auto)
  • Color: (Defaults to $slate) One color is set for the component affecting the background (2% opacity), and the border (40% opacity)
  • Button: Include Small Text button (only for use when it is a single line message)
